Support for writing {@link org.jfree.report.JFreeReport} instances to a character stream in XML format. This package also contains a utility application for converting the old XML format into the new XML format.

The ReportConverter can either be used in command line mode (class org.jfree.report.modules.parser.extwriter.ReportConverter) or with a simple GUI implementation (class org.jfree.report.modules.gui.converter.ReportConverterGUI).
